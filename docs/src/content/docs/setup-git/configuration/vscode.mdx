title: Visual Studio Code(オプション)
---

このページでは、Gitの操作をより便利にするための[Visual Studio Code](https://code.visualstudio.com/)(VSCode)用の拡張機能を紹介します。

導入は**任意**ですが、GUIでの操作性の向上にきっと役立つでしょう。

## Git Graph

https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph

![Git Graph](https://github.com/mhutchie/vscode-git-graph/raw/master/resources/demo.gif)

Gitのコミット履歴をグラフィカルに表示する拡張機能です。コミットの親子関係やブランチの分岐を理解するのに便利です。（上のGIFは[公式リポジトリ](https://github.com/mhutchie/vscode-git-graph)より）

## GitLens

https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens

![GitLens](https://raw.githubusercontent.com/gitkraken/vscode-gitlens/main/images/docs/hovers-current-line.png)

Gitの情報をエディタ上に表示する拡張機能です。コードがいつどのように変更されたかを把握するのに便利です。（上の画像は[公式リポジトリ](https://github.com/gitkraken/vscode-gitlens)より）
