プロキシ環境下でリモートリポジトリへアクセスするには、OSへの設定とは別に、Gitに対して個別のプロキシ設定が必要です。
また、環境外で作業を行う場合は、プロキシ設定を解除する必要があります。

## 設定

```bash
git config --global http.proxy プロキシURL
```

```bash
git config --global https.proxy プロキシURL
```

:::note[DSG関係者向け情報]
内部資料に記載されているプロキシURLを設定してください。
:::

## 設定解除

```bash
git config --global --unset http.proxy
```

```bash
git config --global --unset https.proxy
```
